## v0.1.28 — 請 AI Agent 幫你寫指標和策略

### AI Agent 新能力（桌面版專屬）
- **開口就能生指標**：跟 Agent 說「幫我寫一個量能加權動能指標」，它會直接寫好存進「指標 → 自訂指標」，點選即可加到 K 線上；也能請它修改既有自訂指標（保留你調過的顏色線型）。
- **策略也一樣**：描述進出場邏輯，Agent 寫好回測策略存進「策略回測」面板，選了就能跑；存檔時回報訊號在樣本資料上的觸發次數，太鈍或太敏感一眼看出。
- **同一套安全網**：Agent 寫的程式碼走與手寫相同的沙箱驗證 — 語法錯誤、無窮迴圈（2 秒逾時）都會擋下並讓 Agent 自行修正後重存；成功建立會發通知卡告知。

### 修正
- **Windows**：伺服器管理的背景輔助程序（netstat/taskkill/powershell）不再閃出黑色命令列視窗。
