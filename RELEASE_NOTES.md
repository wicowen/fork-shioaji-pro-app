## v0.1.30 — 首次啟動引導頁 ＋ AI Agent 更會用手

### 首次啟動引導
- **無 API Key 時顯示完整引導頁面**，取代「假裝正常」的空白面板 — 帶著申請 API Key、填入金鑰、選擇 AI 供應商走完整套設定。
- 引導頁側邊欄整合 **AI 助理**協助申請 API Key，登出也會回到引導頁。
- 支援**從 `.env` 檔案匯入 API 金鑰**（改選資料夾方式，原生對話框看不到隱藏檔的限制已繞開）；`.env` 掃描改用萬用字元，不管檔名開頭 `.env` 後面接什麼都找得到。
- AI 助理在引導情境下限縮到對話/設定分頁，預設吃 codex 訂閱，減少初次設定的選項噪音。

### AI Agent（桌面版專屬）
- **新增瀏覽器自動化工具**（agent-browser CLI）— 密碼與驗證碼欄位強制拒填，安全邊界不因新工具鬆動。
- agent 執行 `shioaji` CLI 時**自動對準 app 內建 sidecar 的 port**，不再誤連或另外啟動第三個 daemon。
- 補上申請永豐 API Key 的內建技能、`initialPrompt` 支援。

### 其他修正
- 策略回測時框選單補回遺漏的「1 分 K」。
- 版本驗證補上被動採用路徑，避免誤判正常的舊版 server。

⚠ 回測結果基於歷史資料與簡化成本假設，不代表未來績效；AI 分析僅供參考；自動下單請自行評估風險，盈虧自負。
