為當沖與短線而生的下單體驗

每個功能都針對台股交易流程設計，全部以公開的 Shioaji API 實作 — 股票、期貨、選擇權

⚡

閃電下單

價格梯自動跟隨現價、永不丟失行情。點買賣量即下限價單、在途委託點擊即刪、今日成交標示於實際價位，漲跌停為捲動邊界。

📈

圖表點價下單

K 線上直接點價買賣，委託線顯示於圖上 — 拖曳改價、一鍵取消。支援 MA／EMA／布林通道／VWAP 指標疊圖。

🛡

停損停利與風控

停損／停利觸價單、OCO 括弧單、進場自動掛保護。風控中心一鍵鎖定：單筆口數上限、當日虧損上限、全部刪單。

🗂

多面板自由版面

14 種面板自由拖拉組合：五檔、成交明細、分價量表、選擇權 T 字、委託簿熱圖、行情回放。多組版面可儲存，面板可彈出獨立視窗。

☁️

自選清單雲端同步

自選清單存於 Shioaji 帳戶，跨裝置同步。代碼自動判別股票／期貨／指數，排行榜漲跌幅、量、額一鍵切換。

🖥

桌面級整合

內建 shioaji server（免裝 Python）、伺服器管理與健康監控、系統匣常駐、斷線自動重連重訂閱、自動更新。

🎨

主題與紅綠漲跌

深色／午夜／淺色三種主題，紅漲綠跌或綠漲紅跌自由切換，成交閃動只在真實成交時觸發。

⌨️

鍵盤優先

指令面板 ⌘K 快速切換商品與功能，下單熱鍵、Esc 安全鎖定，操作不離鍵盤。

📊

帳務與損益

多帳戶持倉、委託、帳務即時檢視，未實現損益分析、權益數與保證金、交割款試算，持倉一鍵平倉／反手。

客製你自己的交易介面

Fork 之後對 agent 開口就好：加一個面板、改閃電下單的行為、接上你的策略訊號 — 不用從零刻 UI，這裡已經把地基打完。

為交易 Agent 留好了路

佛心開源

fork 它、拆它、只抄你需要的那一段都歡迎 — 全部程式碼都是學習素材，搭配 Shioaji 官方文件 一起服用。

用自己的 Claude Code／Codex 開始 — 裝上 shioaji skill 就緒：

# Claude Code
/plugin marketplace add Sinotrade/Shioaji
/plugin install shioaji@sinotrade
# 然後直接說：「幫我用 shioaji 做一個 ___」
