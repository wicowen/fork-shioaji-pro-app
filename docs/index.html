Execution

當沖等級的下單武器庫

股票、期貨、選擇權 — 每一種下單方式都為速度與容錯而生

Flash Order

閃電下單：點下去的瞬間，單已經出去了

價格梯自動跟隨現價、永不丟失行情，漲跌停就是捲動邊界。 點買賣量即下限價單、在途委託點擊即刪、今日成交直接標在實際價位上。

  • 持倉與損益常駐頂端 — 部位、均價、未實現一眼掌握
  • 閃電全開 — 自選清單前 N 檔一鍵彈出獨立視窗，8 長條／6 長條自動平鋪
  • 市價買賣／平倉／全刪 一排快捷鍵，Esc 安全鎖定

Grid Orders

鋪單：一次鋪一排，還會跟著價格走

設定檔距、檔數、間隔、每檔量，一鍵鋪出整排限價單。 開啟動態跟隨後價格移動會自動撤舊補新，永遠維持與現價的相對距離 — 速率限制與重複下單防護都想好了。

  • 組合單＋到價監控 — 期權兩腳價差，合成價到目標自動以 IOC 出手
  • 停損停利／OCO 括弧單 — 進場自動掛保護，觸價即動
  • 風控中心 — 單筆口數上限、當日虧損上限、一鍵鎖定全部刪單

Market View

看盤的每一個角度

從大盤熱力到單一選擇權部位的損益曲線，全部即時

Sector Heatmap

熱力圖：今天的錢往哪裡流，一眼看完

整個類股攤成一張熱力圖，色深就是漲跌幅、排序就是成交額。 點任何一格直接連動全終端 — 圖表、五檔、下單面板瞬間切換到那檔股票。

  • 多條件排行榜 — 漲幅／跌幅／量／額一鍵切換，可複選條件
  • 自選清單小線圖 — Canvas 即時 sparkline 看走勢，雲端同步跨裝置
  • 委託簿熱圖・分價量表・行情回放 — 14 種面板自由拖拉組合

Options

選擇權：持倉損益曲線即時畫給你看

選擇權持倉自動畫出結算損益曲線，最大獲利／最大虧損／損益兩平直接標出。 進場前先加模擬部位，跨式、勒式、價差先畫再下。

  • 選擇權 T 字報價 — 全月份快照、ATM 自動定位、點擊連動下單
  • K 線點價下單 — 圖上直接點價買賣、委託線拖曳改價、MA／EMA／布林／VWAP 疊圖
  • 帳務全景 — 多帳戶持倉（含零股）、權益數保證金、資產分布圖、一鍵平倉反手
